# Support

## What this project aims to provide

This repository aims to provide:

- reproducible examples;
- practical setup guidance;
- beginner-friendly documentation;
- a deterministic anonymization utility for Kerio syslog text files;
- reference material for the Kerio Connect Monitoring & Logging project family.

## How to get help

Please use GitHub Issues for:

- bug reports;
- documentation gaps;
- setup problems;
- reproducibility issues;
- feature suggestions.

## Before asking for help

Please include:

- operating system and version;
- Python version;
- the exact step you are on;
- the exact command you ran;
- the exact error message;
- a sanitized sample input if useful;
- relevant logs or screenshots.

## Best-effort support

Support is provided on a best-effort basis.

## Documentation language

- `README.md` is the canonical English README.
- `README.ru.md` is the Russian quick-onboarding README.
- GitHub Release Notes stay in English.

## Important

Please do not post:

- real credentials;
- real private email data;
- non-anonymized production logs;
- proprietary vendor artifacts that cannot be legally redistributed;
- private mapping files unless they have been reviewed and are intentional to publish.
